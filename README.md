# Mortgage Loan Default Risk Analytics

## Overview
This project builds an end-to-end loan-level mortgage risk analytics framework 
to predict next-period default risk using borrower, loan, behavioral, and 
macroeconomic data.

## Data Sources
- Freddie Mac Single-Family Loan-Level Dataset
- Macroeconomic indicators (Housing Price Index, CPI, Unemployment, Recession Index)

## Methodology
- Data cleaning and loan lifecycle alignment
- Rolling time-series feature engineering
- Binary default classification
- Interpretable and tree-based models

## Models
- Logistic Regression
- XGBoost
